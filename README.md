# Douglas's Wordle (Command Line Interface Program)
Currently working on my own version of Wordle in C#.

## Currently at Version 0.3
Includes:
	-  Base Wordle game mode
	-  A developer mode for my own convenience
	-  Some sounds

Plan to Include:
	-  A stats menu
	-  An about menu

How to install:
	1 - Go to the latest release
	2 - Install the ZIP file and extract it
	3 - Run the wordle.exe file to play
